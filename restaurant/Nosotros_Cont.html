EQUIPO DIGGY

OBJETIVO

Desarrollar una aplicación web y móvil que sirva como cliente a un sistema de pedido de comida a domicilio, así como de mandados, proporcionando un sistema intuitivo que agilice el proceso de pedido para el usuario con el fin de satisfacer la demanda de servicios domicilio. Diseñar la arquitectura de la aplicación, el modelo de base de datos y la interfaz de las aplicaciones web y móvil. Proporcionar los servicios de comida y mandados a través de la aplicación móvil y web.


ALCANCE

Para la primera fase del proyecto, se planeó que se enfocara en brindar el servicio comidas a domicilio y su función adicional de mandaditos, únicamente en la zona metropolitana de Tulancingo. Abarcando asi únicamente los municipios de Tulancingo de Bravo, Santiago Tulantepec y Cuautepec.
Como entregables para la finalización de esta primera fase de desarrollo, se presentará la documentación del proyecto, un sitio web y una aplicación móvil que integraran pagos como función adicional.
